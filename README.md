# S Download Manager

A download manager for chrome browser.

## Install

https://chrome.google.com/webstore/detail/s-download-manager/niofdpkjkmgemjmcjlochgpkbbiaipdo

## Development

    npm run dev chrome
    npm run dev firefox
    npm run dev opera
    npm run dev edge

## Build

    npm run build chrome
    npm run build firefox
    npm run build opera
    npm run build edge
